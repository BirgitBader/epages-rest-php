\ep6RESTClient

This is the pure REST client. It is used in a static way.

To connect or reconnect use: RESTClient::connect(HOSTNAME, SHOPNAME, AUTHTOKEN, ISSSL);

To send a command use: RESTClient::send("products");

To change the request method use: RESTClient::setRequestMethod("GET");
